Objetivo
Crear una interfaz web funcional para una tienda e-commerce, permitiendo navegar entre secciones, visualizar productos y acceder al detalle de cada uno.
El proyecto sirve como base para comprender estructura, estilos, interacción y adaptabilidad (responsiveness) en un entorno web realista.

Tecnologías usadas
HTML5 — estructura del sitio.
CSS3 — diseño, estilos y layout responsive.

Ejecutar el proyecto
Opción 1: abrir `index.html` en el navegador.
Opción 2 (recomendada): usar una extensión de servidor local (Live Server en VS Code).

Funcionalidades principales
Listado de productos organizados en tarjetas con imagen, nombre, precio y botón de compra.
Página de detalle con información ampliada del producto.
Footer informativo con datos de la tienda.
Diseño adaptable para móviles, tablets y escritorio.
